# Varun Kumar Growth Portfolio

Static portfolio website for positioning Varun Kumar as a Digital Growth & AI Automation Professional.

The homepage has been rewritten to support:

- Higher-paying growth marketing and digital marketing job opportunities
- Freelance client acquisition
- AI automation and marketing automation consulting
- Personal authority as a future-ready growth systems builder

## Core files

- `index.html` - full homepage structure and rewritten website copy
- `style.css` - responsive visual design and section styling
- `script.js` - navigation, tracking placeholders and contact form behavior
- `assets/images/README.md` - recommended image replacement guide

## Preview locally

Open `index.html` directly, or run a static server from the repo root:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy on GitHub Pages

Open repository Settings, select Pages, choose Deploy from a branch, then select `main` and `/(root)`.

Expected public URL:

`https://digidoxvarun-wq.github.io/varun-growth-portfolio/`

## Tracking and form notes

The `trackEvent(eventName, data)` function currently logs to the browser console. Connect it to GTM, GA4, Meta Pixel or another analytics setup after consent requirements are finalized.

The form validates required fields and opens the visitor's email client using `mailto:`. For higher conversion, connect the form to Formspree, a CRM, or Calendly routing.
